# Changelog

## [0.2.0] - 2024-07-25

### Changed
- Updated the application to use `Doelen`, `Scholen`, and `Vakken` tables from Airtable.
- Removed the old `blog-posts` and `categories` pages and configurations.

### Added

- Initial setup of the Refine application.
- Connected the application to Airtable using a data provider.
- Moved Airtable credentials to environment variables for better security.
- Created documentation folder with architecture.md and changelog.md.
